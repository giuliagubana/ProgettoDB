DESCRIZIONE DELLE ASSOCIAZIONI

USUFRUISCONO

MIN MAX
DIRETTA 0 N
INVERSA 0 N

Gli atleti usufruiscono (opzionale) del noleggio. Il noleggio è usufruito (opzionale) dagli atleti.

TIPO: (0,N)



PARTECIPANO

MIN MAX
DIRETTA 1 N
INVERSA 1 N

Gli atleti partecipano ad una o più lezioni. Le lezioni sono partecipate da uno o più atleti.

TIPO: (1,N)



SOSTENGONO

MIN MAX
DIRETTA 0 N
INVERSA 1 N

I maestri sostengono nessuna o più lezioni. Le lezioni sono sostenute da uno o più maestri.

TIPO: (1,N)



OSPITANO

MIN MAX
DIRETTA 0 N
INVERSA 1 N

Le piste ospitano nessuna o più lezioni. Le lezioni sono ospitate da una o più piste.

TIPO: (1,N)



RISALGONO

MIN MAX
DIRETTA 1 N
INVERSA 1 N

Gli impianti risalgono una o più piste. Le piste sono risalite da uno o più impianti.

TIPO: (1,N)



NECESSITANO

MIN MAX
DIRETTA 1 N
INVERSA 1 N

Gli impianti necessitano di uno o più skipass. Gli skipass sono necessitati da uno o più impianti.

TIPO: (1,N)
